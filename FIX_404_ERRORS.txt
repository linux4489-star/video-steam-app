âŒ ERROR 1: "404 on /" (Main page not found)

CAUSE: public/index.html not being served

SOLUTIONS:

A. Check public/ folder exists:
   - public/index.html
   - public/app.js
   - public/styles.css

B. Check api/index.js has this code:
   ```javascript
   const publicDir = fs.existsSync(path.join(__dirname, '..', 'public'))
     ? path.join(__dirname, '..', 'public')
     : path.join(__dirname, 'public');
   
   app.use(express.static(publicDir));
   
   app.get('*', (req, res) => {
     res.sendFile(path.join(publicDir, 'index.html'));
   });
   ```

C. If on Vercel, check vercel.json:
   ```json
   {
     "version": 2,
     "builds": [
       { "src": "api/index.js", "use": "@vercel/node" }
     ],
     "routes": [
       { "src": "/(.*)", "dest": "api/index.js" }
     ]
   }
   ```

D. Restart server:
   - Press Ctrl+C to stop
   - Run: node api/index.js
   - Try: http://localhost:3000

âŒ ERROR 2: "404 on /api/health" or other API endpoints

CAUSE: Routes not working

SOLUTIONS:

A. Check api/index.js has these routes:
   - /api/health
   - /api/login
   - /api/signup
   - /api/upload
   - /api/videos
   - /api/user

B. Make sure routes are BEFORE catch-all route:
   ```javascript
   // Routes should be here
   app.post('/api/login', ...);
   app.post('/api/signup', ...);
   
   // Catch-all should be LAST
   app.get('*', (req, res) => {
     res.sendFile(path.join(publicDir, 'index.html'));
   });
   ```

C. Test with curl:
   ```powershell
   curl http://localhost:3000/api/health
   ```

D. Check browser network tab (F12 â†’ Network)
   - See what URL is being called
   - See the response status

âŒ ERROR 3: "404 on /api/upload" or file upload

CAUSE: Upload endpoint not working

SOLUTIONS:

A. Check multer is configured:
   ```javascript
   const uploadDir = path.join(__dirname, '..', 'uploads', 'videos');
   
   const storage = multer.diskStorage({
     destination: (req, file, cb) => {
       if (!fs.existsSync(uploadDir)) {
         fs.mkdirSync(uploadDir, { recursive: true });
       }
       cb(null, uploadDir);
     },
     filename: (req, file, cb) => {
       cb(null, `${uuidv4()}-${file.originalname}`);
     }
   });
   
   const upload = multer({ storage });
   ```

B. Create uploads directory:
   ```powershell
   mkdir uploads/videos -Force
   ```

C. Check upload route exists:
   ```javascript
   app.post('/api/upload', authenticateToken, upload.single('video'), ...);
   ```

âŒ ERROR 4: "404 on static files" (CSS, JS not loading)

   ```html

   <script src="public/app.js"></script>  <!-- WRONG -->
   ```
